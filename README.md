# fredilandia
Site feito com um simples jogo interativo
Nesse jogo o internauta deve percebe que existe uma bolinha caindo do topo da tela e mudando sua cor a cada quicada.
Você precisará clicar nas barras (parte inferior da tela) para mudar sua cor conforme a cor da bolinha muda simultaneamente.
Ou seja: a cor da barra deve ser a cor da bola.
